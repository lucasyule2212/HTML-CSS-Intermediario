Legenda dos tipos de display:

Display Block:
  • Utiliza de toda a extensao da página
  • Pode alterar o comprimento e altura (width e height)
  • Ex: Div e Parágrafos
  • Nao funciona dentro de Divs ou Parágrafos, pois quebra o fluxo do "block"
Display Inline:
  • Nao é possível alterar a altura ou comprimento(width e height)
  • Ex: Span
  • Funciona dentro de Divs ou Parágrafos, pois nao quebra o fluxo do "block"
 Display Inline-Block:
  • Mantém o comportamento de Display Inline(Ex: span), porém pode alterar altura ou comprimento(width e height)
  • Ex: Img
  • Funciona dentro de Divs ou Parágrafos, pois nao quebra o fluxo do "block"

Position:

  • Static
    • Posicionado no Flow normal da page.
  • Relative
    • Posicionado no Flow normal da page.
    • Pode ser aplicado modificaçoes(Left,Right,Top,Bottom)
  • Fixed
    • Posicionado de acordo com a localizaçao indicada em relaçao a page inteira(Left,Right,Top,Bottom)
  • Absolute
    • Posicionamento em referencia ao elemento PAI.
    • Pode ser aplicado modificaçoes
  • Sticky
    • Posicionamento em referencia ao elemento PAI.
    • Pode ser aplicado modificaçoes.
    • O elemento "sticky" nao sai da tela durante o scroll(ex:barra de menu de site).

Responsividade e Viewport:

    Características de projeto para uma página responsiva:
  • Medidas dos elementos em referencia aos elementos pai!(Usar tais medidas em porcentagem)
  • Para textos é recomendável o uso da medida "VW"
    • Ex:
    20% da page(PAI=body)
    80% da div pai(PAI=div card1)
    70% da div pai
    (PAI=div card2)

Media Queries:

  • Utilizado para desenvolver a responsividade do layout da page em diferentes devices.
  • Utiliza "@media screen and (max-width: limiteDePixels)"
  • Mobile First: Fazer o layout da page para mobile primeiro, depois usar o Media para fazer a page responder em devices maiores.
